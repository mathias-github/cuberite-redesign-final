If you feel like contributing to Cuberite, there are various ways you can do so.

Code

We welcome any interested developers to contribute to Cuberite. Check out the Getting Started guide for more information about development.

Getting Started

Report Bugs

Bug reports help us keep track of what should be added to or fixed in the Cuberite server. If you've found a bug, or a feature is missing, feel free to report it in the bug tracker.

Bug Tracker

Donate

If coding isn't for you, you may also donate to help Cuberite. Donations will be used to cover web hosting expenses and reward developers.
