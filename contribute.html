If you feel like contributing to the Cuberite project, there are various ways you can do so.

Code

Cuberite is licensed under the Apache License V2, and we welcome any interested developers to contribute to the project. Check out the Getting Started guide for more information about development, and visit our GitHub repository to start developing.

Getting Started Code Repository

Report Bugs

Bug reports help us keep track of what should be fixed on or added to the Cuberite server. If you've found a bug or a missing feature, feel free to report it in the bug tracker. Remember to use the search function before creating a new bug report, so we can avoid duplicate bug reports.

Bug Tracker

Donate

You may also donate to help the project. Donations will be used to cover web hosting expenses and to reward developers.
