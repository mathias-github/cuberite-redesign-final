Plugins let you add new gameplay features to your server and modify existing features.

Install Plugins

There are many plugins available to use, and installing them is easy. We have a plugin repository where you can find most plugins available for Cuberite. The user's manual contains instructions on how to install a plugin.

Installation Guide Download Plugins

Develop Plugins

Although there is a large amount of plugins to choose from, you may not always find exactly what you want. In this case, you can create a plugin request in our forum, or create a plugin yourself. Information about plugin development can be found in the API documentation.
